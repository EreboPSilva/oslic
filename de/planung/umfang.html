Der Scope vom OSLiC

Die Deutsche Telekom AG (DTAG) entwickelt ein Open Source License Compendium, das der gesamten Open Source Community zugänglich gemacht werden soll...

Für große (IT-) Firmen bildet die Menge der Open Source Lizenzen eine besondere Herausforderung. Es ist einfach zu teuer, wenn sie immer wieder in jedem ihrer vielen, von Open Source betroffenen Projekte jemand als Open Source Lizenzexperten auswählen und ausbilden müssen, nur um sicherzustellen, dass sie sich als Firma lizenzgerecht verhalten. Besser wäre es, wenn sie bereits so etwas vorfänden, wie ...

... ein  "Kompendium", das für jedes Nutzungsszenario und für jede wichtige Open Source Lizenz eine Art To-Do-Liste anbietet, die beschreibt, was man tun muss, um diese Open Source Lizenz hinsichtlich dieses Nutzungsszenarios zu erfüllen.

Im Sinne der Open Source Software Bewegung möchte die DTAG dieses Kompendium unter der 'Creative Commons Attribution-ShareAlike 3.0 Germany Lizenz' veröffentlichen. Und mehr noch, die DTAG will sich das gängige Prinzip zu eigen machen, ...

... 'früh und oft zu veröffentlichen', um das Kompendium zusammen mit der Open Source Community zu entwickeln.
