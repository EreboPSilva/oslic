Installing the OSLiC development environment

For "installing" the OSLiC sources you need to do nothing more than extracting the zip file (if you pressed the zip button on the OSLiC Github frontend) and changing into the working directory. There you will find the following directory structure:

  • directory bibfiles :- contains the BiBTeX files for managing the secondary literature.
  • directory bin :- will contain some meta scripts (if necessary later on).
  • directory btexmat :- contains documents to configure and test the styling of the BibTeX files when they are included into the LaTeX files.
  • file CHANGES :- documents major development steps.
  • directory extracts :- contains abstracts and summries of some of the evaluated secondary literature.
  • directory graphics :- contains pictures and their eps derivatives being embedded into the LaTeX files.
  • file Makefile :- is the important compiling tool by which the LaTeX based OSLiC sources can be converted / compiled into dvi-, pdf-, or rtf-files.
  • directory mindmaps :- contains some tools for organizing the mental work.
  • file oslic-de.tex :- is the German version of the Open Source License compendium. (Note: the OSLiC will firstly be written in English and later on translated into German.)
  • file oslic-en.tex :- is the English version of the Open Source License compendium.
  • file README :- contains the OSLiC definition for Github.
  • file release.tex :- contains the current release for being included into the LaTex files.
  • directory snippets :- contains directories and includable LaTeX files for each chapter, section, and part of the OSLiC.
  • directory templates :- contains some templates.
  • directory tmp :- is needed by some executing editor.
