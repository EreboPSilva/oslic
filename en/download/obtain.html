OSLiC @ Github

https://github.com/dtag-dbu/oslic

Obtaining the OSLiC Repository (participate permanently | frequently)

  • You can fetch the OSLiC repository by using the methods mentioned on the OSLiC Github frontend, particularly by the commands
    • git clone https://github.com/dtag-dbu/oslic.git
    • git clone git@github.com:dtag-dbu/oslic.git

Obtaining the OSLiC Repository (participate  frequently)

  • Naturally you might also fork the OSLiC repository in the spirit of Github. We will send the pull requests at the most important moments.

Obtaining the OSLiC Files (participate case-by-case)

  • You can fetch the OSLiC files as zip archive by pressing zip button of the OSLiC Github frontend, or by directly calling the url
    • https://github.com/dtag-dbu/oslic/zipball/master
