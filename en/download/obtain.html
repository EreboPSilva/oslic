OSLiC @ Github

https://github.com/telekom/oslic

Obtaining the OSLiC Repository (participate permanently | frequently)

  • You can fetch the OSLiC repository by using the commands
    • git clone https://github.com/telekom/oslic.git
    • git clone git@github.com:telekom/oslic.git

Obtaining the OSLiC Repository (participate  frequently)

  • Of course, you might also fork the OSLiC repository in the spirit of Github. We will send the pull requests at the most important moments.

Obtaining the OSLiC Files (participate case-by-case)

  • You can fetch the OSLiC files as zip archive by pressing zip button of the OSLiC Github frontend, or by directly calling the url
    • https://github.com/telekom/oslic/zipball/master
